디스크 관리 & LVM 정리

2026-04-19

[디스크 관리]

fdisk /dev/sd
 : 특정 디스크 직접 편집 (파티션 생성/삭제)

fdisk -l /dev/sdc
 : 읽기 전용으로 파티션 정보 조회

lsblk
 : 전체 블록 디바이스 구조를 트리 형태로 확인


[LVM 생성 순서]

1. PV (Physical Volume)
   (1) 조회
       pvs
       pvdisplay

   (2) 생성
       pvcreate /dev/sd?
       예) pvcreate /dev/sd1


2. VG (Volume Group)
   (1) 조회
       vgs
       vgdisplay

   (2) 확장
       vgextend vg명 /dev/sd?
       (기존 VG에 디스크 추가, 포맷 X)
       예) vgextend vg01 /dev/sdc1

   (3) 생성
       vgcreate vg명 /dev/sd?
       (새 VG 생성, 포맷 필요)
       예) vgcreate vg01 /dev/sdc1


3. LV (Logical Volume)
   (1) 조회
       lvs
       lvdisplay

   (2) 확장
       lvextend -l +100%FREE /dev/vg명/lv명
       (기존 LV 확장, 포맷 X)

   (3) 생성
       lvcreate -l +100%FREE -n lv명 vg명
       (새 LV 생성, 이후 파일시스템 생성 필요)


4. 파일시스템 반영
   resize2fs /dev/vg01/app
   (ext4 기준)
